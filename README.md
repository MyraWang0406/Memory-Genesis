# Memory Genesis: Life Coach & Decision Revise Agent System

**Memory Genesis** is a multi-agent system designed for personal growth, reflection, and decision-making optimization. It helps you uncover repeated patterns in past regrets, grasp the drivers behind your choices, and align your future actions with your true goals.

> "Unresolved patterns loop until you choose anew. Unhealed emotions escalate until you shift your attitude."

---

## 🌟 Core Philosophy

We believe that many life challenges stem from "inertia path dependency." By systematically recording key events, analyzing stakeholder perspectives, and calibrating ultimate goals, Memory Genesis empowers you to turn passive experiences into active wisdom.

## 🛠 Key Features

### 1. Retrace the Thread (Before)
*   **Record Context**: Document key events, people, and situational details.
*   **Emotional Recall**: Retrace the thread of what happened and explore if similar historical patterns or unresolved feelings exist.

### 2. Understand Structure (Now)
*   **Factor Analysis**: Clarify key factors and how they interact with others.
*   **Stakeholder Mapping**: Map the stakeholders, relationships, and entangled stakes involved in any situation.
*   **Perspective Simulation**: Simulate perspectives from participants, stakeholders, and neutral observers to gain a 360-degree view.

### 3. Conduct Align (After)
*   **Goal Calibration**: Look ahead to what you truly want and align your actions accordingly.
*   **Motivation Clarification**: Clarify intent, drivers, and ultimate goals.
*   **Action Optimization**: Optimize short-term actions while striving for long-term strategic goals.

---

## 🔄 Workflow

1.  **Input Event**: Record what happened in your daily life.
2.  **Look Back**: Recall similar past experiences and emotional triggers.
3.  **Understand**: Analyze roles, motives, and the underlying structure of the event.
4.  **Calibrate**: Clarify your true desires and ultimate objectives.
5.  **Respond**: Formulate optimized next steps and future guidelines.

## 🚀 Getting Started

### Prerequisites
*   Node.js (v18+)
*   pnpm or npm

### Installation
```bash
# Clone the repository
git clone https://github.com/MyraWang0406/Memory-Genesis.git

# Install dependencies
pnpm install

# Start development server
pnpm dev
```
